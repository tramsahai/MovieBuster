HOW TO CALCULATE AVERAGE RATING:
    //Store movie data in parallel arrays
    movie_titles = ["The Godfather", "The Dark Knight", "Pulp Fiction"]
    directors = ["Francis Coppola", "Christopher Nolan", "Quentin Tarantino"] 
    ratings = [5.0, 5.0, 4.0]
    comments = ["Masterpiece", "Epic superhero", "Quirky and fun"]
    
    total = 0
    // Loop through all ratings and add them up
    FOR each rating IN ratings:
        total = total + rating
    
    // Calculate average by dividing total by number of movies
    average = total / number_of_movies
    DISPLAY "Average Rating: 4.7/5"

WHEN USER TAPS "ADD MOVIE" BUTTON:
    GO TO ADD MOVIE SCREEN

WHEN USER TAPS "VIEW MOVIES" BUTTON:
    IF there are movies to show:
        GO TO MOVIE LIST SCREEN
    ELSE:
        SHOW MESSAGE "No movies available"

// PART 3: 

ADD MOVIE SCREEN HAS:
    - Text box for movie title
    - Text box for director name  
    - Text box for rating (1-5)
    - Text box for comments
    - "Save" button
    - "Cancel" button

WHEN USER TAPS "SAVE":
    // First check if all information is valid
    IF movie_title is empty:
        SHOW ERROR "Please enter movie title"
    ELSE IF director is empty:
        SHOW ERROR "Please enter director name"
    ELSE IF rating is empty OR not between 1-5:
        SHOW ERROR "Please enter rating between 1-5"
    ELSE IF comments is empty:
        SHOW ERROR "Please enter comments"
    ELSE:
        // All information is good, save movie
        ADD new movie to our arrays:
            ADD movie_title to movie_titles array
            ADD director to directors array  
            ADD rating to ratings array
            ADD comments to comments array
        
        SHOW MESSAGE "Movie added successfully!"
        GO BACK TO MAIN SCREEN

WHEN USER TAPS "CANCEL":
    GO BACK TO MAIN SCREEN
